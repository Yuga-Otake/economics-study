## 1. 経済指標のマインドマップ

```
                                     ┌─── 名目GDP
                                     │
                             ┌───── GDP ──── 実質GDP ─── GDPデフレーター
                             │       │
                             │       └─── 三面等価 ─┬─ 生産面
                             │                     ├─ 分配面
                             │                     └─ 支出面
                             │
                ┌─── 国民経済計算 ─┼─── GNI
                │            │
                │            └─── 国民所得
                │
                │                    ┌─── 消費者物価指数(CPI)
                │            ┌─── 物価指数 ─┼─── 企業物価指数(PPI)
                │            │              └─── GDPデフレーター
経済指標 ───────┼─── 各種経済統計 ─┤
                │            │              ┌─── 完全失業率
                │            │      ┌─── 雇用統計 ─┤
                │            │      │              └─── 有効求人倍率
                │            └─────┤
                │                   │      ┌─── 鉱工業生産指数
                │                   └─── 生産統計 ─┤
                │                          └─── 第3次産業活動指数
                │
                │                          ┌─── 先行指数
                │            ┌─── 景気動向指数 ─┼─── 一致指数
                │            │                  └─── 遅行指数
                └─── 景気指標 ─┤
                             │      ┌─── 日銀短観
                             └─── 景況感 ─┤
                                          └─── 消費者態度指数
```
