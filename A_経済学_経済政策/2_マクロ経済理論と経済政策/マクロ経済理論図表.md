## 1. IS-LM曲線と財政・金融政策

```
                   利子率(r)
                      ↑
                      │
                      │                          LM
                      │                      ／
                      │                   ／
                  r2 ─┼───────────────●─────
                      │               ／|      
                  r1 ─┼───────●─────│──────
                      │       ／      |      
                      │    ／         |               IS
                      │ ／            |              
                      │               |
                      │               |
                      └───────────────┼───────────→ 所得(Y)
                                     Y1      Y2
```

**財政政策**: IS曲線を右にシフトさせ、所得水準をY1からY2に増加させる。
**金融政策**: LM曲線を右にシフトさせ、利子率をr2からr1に低下させる。

## 2. 総需要・総供給モデル（AD-AS）

```
               物価水準(P)
                     ↑
                     │                       SRAS
                     │                    ／
                 P2 ─┼───────────●────────
                     │            /|          
                 P1 ─┼───●──────│───────
                     │    / |     |      
                     │   /  |     |              LRAS
                     │  /   |     |              |
                     │ /    |     |              |
                     │/     |     |              |
                     └──────┼─────┼──────────────→ 実質GDP(Y)
                           Y1     Y2             Yf
                                       AD1   AD2
```

SRAS: 短期総供給曲線
LRAS: 長期総供給曲線（潜在GDP水準Yfで垂直）
AD: 総需要曲線

総需要の増加（AD1→AD2）により、短期的には実質GDPがY1からY2へ、物価水準がP1からP2へ上昇。長期的には、物価のみが上昇し、実質GDPは潜在GDP水準（Yf）に戻る。

## 4. フィリップス曲線の長期・短期関係

```
               インフレ率
                   ↑
                   │
                   │                        長期フィリップス曲線（垂直）
                   │                            |
                π3 ─┼─────────────────────●────|───
                   │                      ／|   |
                π2 ─┼──────────────●────────|───|───
                   │                 ／    |   |
                π1 ─┼──────●─────────────|───|───
                   │       ／               |   |
                   │   ／                   |   |
                   │／                      |   |
                   └──────────────────────┼───┼────→ 失業率
                                         u*  u1 u2
                     SRPC3     SRPC2      SRPC1
```

SRPC: 短期フィリップス曲線
u*: 自然失業率

短期的には、インフレ率と失業率の間にトレードオフ関係が存在（SRPC曲線上の動き）。
長期的には、インフレ期待の変化により短期フィリップス曲線自体がシフトし、自然失業率u*において垂直な長期フィリップス曲線が形成される。

## 5. 乗数効果

```
                 総所得
                   ↑
                   │
                   │
               ΔY ─┼───────────────────
                   │                   /|
                   │                  / |
                   │                 /  |
                   │                /   |
                   │               /    |
                   │              /     |
                   │             /      |
                   │            /       |
                   │           /        |
                   │          /         |
                   │         /          |
                   │        /           |
              ΔG or ΔI ─┼───────────────────
                   │     |               |
                   │     |               |
                   └─────┼───────────────┼───→ 消費・投資の変化
                         0              時間
                         
            ΔY = k × ΔG  (kは乗数、k = 1/(1-MPC))
```

政府支出やその他自律的支出の変化（ΔG or ΔI）が乗数効果を通じて、より大きな総所得の変化（ΔY）をもたらす。
乗数の大きさは限界消費性向（MPC）に依存し、k = 1/(1-MPC)で計算される。

## 6. 開放経済のIS-LM-BP曲線

```
                  利子率(r)
                     ↑
                     │
                     │                         BP
                     │                      ／
                     │                   ／
                     │                ／
                     │             ／             LM
                     │          ／           ／
                     │       ／          ／
                     │    ／         ／
                     │ ／        ／
                     │       ／                     IS
                     │   ／
                     │／
                     └───────────────────────────→ 所得(Y)
```

IS曲線: 財市場の均衡
LM曲線: 貨幣市場の均衡
BP曲線: 国際収支の均衡

固定為替レート制度下では、IS-LM均衡点がBP曲線上または下にある場合、国際収支の不均衡が生じ、中央銀行の外貨準備が変動する。
変動為替レート制度下では、為替レートの調整により国際収支は均衡する（BP曲線上に均衡点が移動）。
