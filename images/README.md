# 経済学・経済政策ノート用画像フォルダ

このフォルダには経済学・経済政策ノートで使用する図表やマインドマップなどの画像ファイルを保存します。

## 必要な画像ファイル
- economic_indicators_mindmap.png - 経済指標の関係性を示すマインドマップ
- gdp_three_aspects.png - GDPの三面等価を図示したもの
- business_cycle.png - 景気循環を表す図
- phillips_curve.png - フィリップス曲線
- is_lm_curve.png - IS-LM曲線
- supply_demand.png - 需要と供給の曲線

## 作成方法
- PowerPointやExcelなどで図表を作成し、PNG形式で保存
- オンラインのマインドマップツールで作成した図を保存
- 手書きの図をスキャンまたは写真撮影して保存
